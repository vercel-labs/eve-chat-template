# Origins

This project started as a fork of `danvoulez/mistral-eve`, which itself was a
transplant of the Dynamic Projection engine and Lab airlock from an internal
runtime into a standalone Next.js app. The transplant progress tracker lives in
`docs/archive/transplante-progresso.md`.

The finishing work (reader/Lab/embedding provider registries, local defaults,
menu surfacing, and docs) is the bridge from "half-transplanted fork" to
"independent, self-running piece open to the world through plugins."
